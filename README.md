INNOVACIONES: 

1. Sistema de combate activo
2. Minimapa del dungeon
3. Sistema de puntuación y contador de turnos

COMPILACION 

Linux
Compilar con Make
make
./dungeon_crawler
Compilar con CMake
mkdir build
cd build
cmake ..
make
./dungeon_crawler

Windows
Compilar con MinGW
mingw32-make
dungeon_crawler.exe
Compilar con CMake
mkdir build
cd build
cmake ..
cmake --build .

macOS
Compilar con Make
make
./dungeon_crawler
Compilar con CMake
mkdir build
cd build
cmake ..
make
